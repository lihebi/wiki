* Job control
- =C-z= to suspend.
- =fg= to bring it to foreground.
- =jobs= to list all background jobs,
  and =kill %1= to kill the first.

* Install fonts
- OTF: OpenType
- TTF: TrueType
- EOT: Embedded Open Type
- WOFF: Web Open Font Format

To install:

#+BEGIN_EXAMPLE
cp *.otf ~/.fonts
sudo fc-cache
fc-list
#+END_EXAMPLE
